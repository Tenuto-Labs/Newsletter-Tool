Tenuto Newsletter Tool

Live frontend:
https://tenuto-labs.github.io/Newsletter-Tool/

This repo now contains only the code required for:
- the hosted web app UI (`index.html`)
- the assistant backend API (`assistant_api.py`)
- weekly newsletter generation/email (`newsletter.py`)
- scheduled newsletter execution (`.github/workflows/newsletter.yml`)

Notes:
- LinkedIn/X scraping is handled in separate repos.
- Generated newsletter output files are ignored by default.
